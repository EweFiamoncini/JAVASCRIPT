Tipos de Comentários JS

Em JavaScript, os comentários são usados para adicionar notas explicativas ao seu código, que são ignoradas pelo navegador e não afetam a execução do programa. Eles servem para tornar o seu código mais fácil de entender, tanto para você mesmo no futuro quanto para outros desenvolvedores.

Existem duas maneiras principais de criar comentários em JavaScript:

  • Comentário de Linha Única = //
    Este é o tipo mais comum. O comentário começa com duas barras e tudo o que vem depois delas na mesma linha é ignorado.

  • Comentário de Múltiplas Linhas = /* ... */
    Este tipo é ideal para escrever comentários mais longos, que ocupam várias linhas. O comentário começa com /* e termina com */. Tudo o que está entre esses dois delimitadores é ignorado.
