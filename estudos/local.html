Introdução ao JavaScript

O JavaScript é uma linguagem de programação usada principalmente para tornar páginas da web interativas e dinâmicas.

Pense na internet como uma casa: o HTML é a estrutura (as paredes e o telhado), o CSS é a decoração (a cor da pintura e os móveis), e o JavaScript é a eletricidade (as luzes que acendem, o ar-condicionado que liga).

Ele permite que você crie coisas como:

  • Menus que se expandem.
  • Formulários que validam informações antes de serem enviados.
  • Animações e carrosséis de imagens.
  • Mapas interativos.

Embora tenha sido criado para a web, o JavaScript hoje é usado em muitos outros lugares, como em aplicativos de celular, jogos e até em servidores de internet. É uma das linguagens mais populares e essenciais para quem trabalha com desenvolvimento web.

Introduzindo JS interno e externo!

  • Interno = quando codigo JS é inserido no meio do codigo html!
  • Externo = quando codigo JS é inserido por meio de um outro arquivo JS!
