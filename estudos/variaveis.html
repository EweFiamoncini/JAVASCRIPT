Tipos de Variáveis JS

  • String = Letras
  • Number = Numeros
  • Boolean = Verdadeiro/Falso

Regras para declaração das variáveis

  • Não podem ser iniciadas com numeros, apenas com letras ou "_"
  • Não podem ser utilizados caracteres especiais como "ç", "^", "~"
  • Não podem ser iguais as palavras reservadas da linguagem. Como:
    • this (referência ao objeto atual)
    • new (cria uma nova instância de um objeto)
    • super (chama o construtor da classe pai)
    • extends (cria uma herança de classe)
    • static (cria métodos e propriedades estáticas)

Declarando variáveis com a instrução "var"

  • var curso
  • var Curso
  • var CURSO

Todas estas variaveis acima sao validas e diferentes!

Exemplo de variaveis

  • var nome, (Válido)
  • var AulaVariaveis, (Válido)
  • var _idade, (Válido)
  • var 123nome, (Inválido). Pois:
    • O nome de uma variável precisa seguir algumas regras específicas. A mais importante delas é que o nome de uma variável não pode começar com um número.
    • Essa regra existe para evitar ambiguidades e garantir que o interpretador do JavaScript (o programa que lê e executa o código) possa distinguir facilmente entre um número e um nome de variável.
  • var número, (Inválido). Pois:
    • Em versões mais antigas do JavaScript, o uso de caracteres especiais ou acentuados em nomes de variáveis é desaconselhado.
