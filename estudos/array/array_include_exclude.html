Array - Método de inclusão e exclusão de elemento

Inclusão:

var lista_estados = Array("Rio de Janeiro", "São Paulo")

//include element end array
lista_estados.push("Santa Catarina")

//include element frist array
lista_estados.unshift("Paraná")

console.log(lista_estados)

*Resultado no Console (F12)*

Exclusão:

var lista_alunos = Array("Joao", "Maria", "Ewerton", "Lucas")

//exclude element end array
lista_alunos.pop()

//exclude element first array
lista_alunos.shift()

console.log(lista_alunos)

*Resultado no Console (F12)*
