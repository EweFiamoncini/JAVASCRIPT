Array - Multidimensional

O que é um Array Multidimensional?

Arrays multidimensionais são arrays que contêm outros arrays, ou seja, são "arrays de arrays". A forma mais comum é o array bidimensional, que pode ser visualizado como uma grade ou uma tabela com linhas e colunas.

Criando um Array Multidimensional:

Para criar um array Multidimensional, você simplesmente aninha arrays dentro de um array principal.

var lista_coisas = Array()
lista_coisas["numeros"] = Array(10, 20, 30, 40, 50)
lista_coisas["string"] = Array("Rio de Janeiro", "São Paulo", "Santa Catarina")
lista_coisas["misto"] = Array("Ewerton", 25, true, "Santa Catarina")

console.log(lista_coisas)

*Resultado no Console (F12)*
