Array - Ordenação

Ordenando Strings e Números (padrão)

Por padrão, o método sort() converte os elementos para string e os ordena de acordo com a ordem alfabetica.
Isso funciona bem para strings, mas pode levar a resultados inesperados com números.

Código:

var lista_pessoas = Array("Caio" , "Joao", "Lucas", "Ewerton")

console.log(lista_pessoas.indexOf("Lucas"))

*Resultado no Console (F12)*
