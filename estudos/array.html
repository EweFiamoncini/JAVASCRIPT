Array - Introdução

Um array (ou vetor) em JavaScript é uma estrutura de dados que armazena uma coleção de elementos. Esses elementos podem ser de qualquer tipo: números, strings, booleanos, objetos, ou até mesmo outros arrays. Eles são indexados numericamente,

Características Principais

  • Coleção de Dados: Arrays servem para agrupar múltiplos valores, como números, textos (strings), objetos ou até mesmo outros arrays.
  • Índices Numéricos: Os elementos são acessados por um índice numérico.
    O primeiro elemento de um array sempre tem o índice 0, o segundo o índice 1, e assim por diante.
  • Tamanho Dinâmico: O tamanho de um array não é fixo. Você pode adicionar ou remover elementos a qualquer momento.
