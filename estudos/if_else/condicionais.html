Operadores de comparação (condicionais)

Também conhecidos como operadores relacionais
Servem para formar expressões condicionais para o comando if

Operadores:

    Igual(==) - Verifica se os valores comparados são iguais

    Igual(===) - Verifica se os valores comparados são iguais e do mesmo tipo

    Igual(!=) - Verifica se os valores comparados são diferentes

    Igual(!==) - Verifica se os valores comparados são diferentes e de tipos diferentes

    Igual(<) - Verifica se o valor da esquerda é menor que o valor da direita

    Igual(>) - Verifica se o valor da esquerda é maior que o valor da direita

    Igual(<=) - Verifica se o valor da esquerda é menor ou igual ao valor da direita

    Igual(>=) - Verifica se o valor da esquerda é maior ou igual ao valor da direita
