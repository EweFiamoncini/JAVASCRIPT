If e Else

Em JavaScript, a instrução if...else é uma das estruturas de controle de fluxo mais fundamentais. Ela permite que você execute diferentes blocos de código dependendo de uma condição ser verdadeira (true) ou falsa (false).

  • if ...

    • A forma mais simples é a instrução if. Se a condição entre parênteses for verdadeira, o código dentro das chaves ({}) será executado.

  • if ... else ...

    • A instrução if...else adiciona uma segunda opção. Se a condição no if for falsa, o código dentro do bloco else será executado.

  • if ... else if ... else

    • Para testar várias condições, você pode usar a instrução else if. O JavaScript verificará cada condição em ordem. Ele executará o bloco de código da primeira condição que for verdadeira e, em seguida, pulará o restante da estrutura if...else if...else.

Exemplos:

if ...

if (condição) {
//trecho de codigo que sera executado caso condição seja verdadeira
}

if ... else

if (condição) {
//trecho de codigo que sera executado caso condição seja verdadeira

} else {
//trecho de codigo que sera executado caso condição seja falsa

}

if ... else if ... else

if (condição) {
//trecho de codigo que sera executado caso condição seja verdadeira

} else if (condição) {
//trecho de codigo que sera executado caso condição seja falsa

} else {
//trecho de codigo que sera executado se nenhum dos valores acima coincidir

}
