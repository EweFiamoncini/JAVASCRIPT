Eventos - Mouse

Eventos de mouse em JavaScript são ações que o usuário realiza com o mouse, como clicar, mover, pressionar ou soltar. Eles são essenciais para criar interfaces interativas, permitindo que você reaja às ações do usuário em tempo real.

  • onclick:
    • Ocorre quando o usuário clica no elemento.
      Clique aqui
  • ondblclick:
    • Ocorre quando o usuário clica duas vezes no elemento.
      Clique aqui
  • onmouseup:
    • Ocorre quando qualquer botão do mouse é clicado sobre o elemento.
      Clique aqui
  • onmouseover:
    • Ocorre quando o ponteiro do mouse se move para dentro dos limites do elemento.
      Coloque o cursor aqui
  • onmouseout:
    • Ocorre quando o ponteiro do mouse se move para fora dos limites do elemento.
      Coloque o cursor aqui
