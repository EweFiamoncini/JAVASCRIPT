Functions(Funções) - Nativas para manipular datas

O JavaScript tem um objeto nativo chamado Date que permite trabalhar com datas e horas. Ele é usado para criar, manipular e formatar informações de data e tempo.

  • new Date(): Cria um objeto Date com a data e hora atuais.
  • getDate(): Obtém o dia do mês (1-31).
  • getMonth() + 1: Obtém o mês (0-11). Como ele retorna o mês começando do 0 precisamos adicionar + 1 após o comando!
  • getFullYear() + 1: Obtém o ano (4 dígitos).
  • toString(): Converte a data em uma string legível, incluindo o fuso horário.
  • setDate(): É um método do objeto Date em JavaScript usado para definir o dia do mês de uma data. Ele aceita um único argumento: um número inteiro entre 1 e 31.
