Functions(Funções) - Escopo de variaveis

O escopo de variáveis em JavaScript define a acessibilidade e a visibilidade de uma variável, ou seja, onde no seu código você pode usá-la. Entender o escopo é fundamental para evitar bugs, colisões de nomes e para escrever código mais limpo e seguro.

Existem três tipos de escopo em JavaScript:

  • Global Scope
    • Uma variável declarada no nível superior do seu script, fora de qualquer função ou bloco ({}), tem um escopo global. Isso significa que ela pode ser acessada de qualquer lugar no seu código, incluindo dentro de funções e blocos.

  • Function Scope
    • Variáveis declaradas dentro de uma função têm um escopo de função. Elas só podem ser acessadas dentro da função em que foram criadas. Se você tentar acessá-las fora, obterá um erro.

  • Block Scope
    • Introduzido no ES6 (ES2015) com as palavras-chave let e const, o escopo de bloco é mais restrito. Uma variável declarada dentro de um bloco de código (delimitado por chaves {}) só é acessível dentro desse bloco. Isso inclui if statements, for loops, while loops, etc.
