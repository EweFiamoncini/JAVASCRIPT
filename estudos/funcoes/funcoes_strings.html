Functions(Funções) - Nativas para manipular strings

Funções nativas para manipular strings são incrivelmente úteis para formatar texto, buscar informações, substituir caracteres e muito mais. Essas funções são métodos que você pode chamar diretamente em qualquer string.

  • length: retorna o número de elementos de uma sequência.

  • charAt(): retorna um caractere em um índice especificado de uma string.

  • indexOf(): é usado para encontrar a posição da primeira ocorrência de um valor especificado dentro de uma string ou array.

  • replace: é usado para substituir uma parte de uma string por outra. Ele retorna uma nova string sem modificar a string original.

  • substr: é usado para extrair uma parte de uma string. Começa em um índice especificado e tem um comprimento determinado.

  • toLowerCase: é uma função de string que retorna uma nova string com todos os caracteres convertidos para minúsculas.

  • toUpperCase: é uma função de string que retorna uma nova string com todos os caracteres convertidos para maiúsculas.

  • trim: é uma função de string que remove os espaços em branco do início e do fim de uma string. Ele retorna uma nova string sem esses espaços.
