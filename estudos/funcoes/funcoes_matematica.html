Functions(Funções) - Nativas para tarefas matemâticas

O JavaScript possui um objeto nativo chamado Math que oferece uma variedade de funções e constantes para realizar tarefas matemáticas comuns. Diferente de outros objetos, o Math não precisa ser instanciado com new. Você pode acessar suas propriedades e métodos diretamente.

  • Math.ceil(x): Arredonda x para cima, para o número inteiro mais próximo.
  • Math.floor(x): Arredonda x para baixo, para o número inteiro mais próximo.
  • Math.round(x): Arredonda x para o número inteiro mais próximo.
  • Math.random(x): Retorna um número pseudo-aleatório de ponto flutuante entre 0 (inclusive) e 1 (exclusivo).
