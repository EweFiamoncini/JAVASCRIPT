DOM JS

DOM em JavaScript, ou o Document Object Model, é a forma como o JavaScript enxerga e interage com uma página da web. Pense nisso como uma árvore genealógica de todos os elementos HTML de uma página.

Cada tag HTML, como html, head, body, div, p, etc., é um nó na árvore do DOM. O JavaScript pode "navegar" por essa árvore para encontrar, modificar, criar ou remover qualquer um desses nós.

O que você pode fazer com o DOM?

Com o DOM, o JavaScript se torna o motor de uma página interativa. Você pode fazer coisas como:

  • Encontrar elementos: Selecionar um elemento específico na página, como um botão ou um parágrafo, usando seus IDs, classes, ou nomes de tags.
  • Modificar elementos: Mudar o conteúdo de um parágrafo, a cor de um título, a imagem de um site, ou qualquer outro atributo.
  • Criar e remover elementos: Adicionar novos elementos à página em tempo real (como uma nova linha em uma tabela) ou remover elementos que já existem.
  • Reagir a eventos: Fazer com que a página responda a ações do usuário, como cliques do mouse, digitação no teclado, ou o movimento do cursor.
