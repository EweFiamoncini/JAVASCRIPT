Functions(Funções) JS

As funções em JavaScript são blocos de código reutilizáveis que realizam uma tarefa específica. Elas permitem organizar e modularizar o código, tornando-o mais fácil de ler, manter e depurar. Pense em uma função como uma receita de bolo: você pode usar a mesma receita várias vezes para fazer bolos diferentes, apenas alterando os ingredientes (os parâmetros). As funções podem aceitar dados de entrada (argumentos) e retornar um valor como saída.
