Eventos JS

Eventos em JavaScript são ações ou ocorrências que acontecem em um sistema. O navegador "escuta" essas ações e você pode usar o JavaScript para responder a elas. Por exemplo, um evento pode ser quando o usuário clica em um botão, digita em um campo de texto, move o mouse, ou quando a página termina de carregar.

A seguir, apresento uma lista com alguns dos eventos mais comuns, divididos por categoria.

  • Eventos de Mouse
    • onclick: Ocorre quando o usuário clica em um elemento.
    • ondblclick: Ocorre quando o usuário clica duas vezes em um elemento.
    • onmouseup: Ocorre quando o botão do mouse é liberado sobre um elemento.
    • onmouseover: Ocorre quando o ponteiro do mouse se move para dentro dos limites de um elemento.
    • onmouseout: Ocorre quando o ponteiro do mouse se move para fora dos limites de um elemento.

  • Eventos de Teclado
    • onkeydown: É disparado no momento em que uma tecla é pressionada.
    • onkeyup: É disparado no momento em que uma tecla é liberada.

  • Eventos de Janela(frame)
    • onresize: Disparado quando a janela do navegador é redimensionada pelo usuário.
    • onscroll: Disparado sempre que o usuário rola a janela.

  • Eventos de Formulario
    • onfocus: É disparado quando um elemento, como um campo de formulário (input, textarea, select), recebe o foco do usuário. Isso acontece quando o usuário clica ou usa a tecla Tab para navegar até o elemento.

    • onblur: É disparado quando um elemento de formulário ou qualquer outro elemento com capacidade de foco perde o foco do usuário. Isso geralmente acontece quando o usuário clica em outro lugar da página ou navega para outro campo usando a tecla Tab.

    • onchange: É disparado quando o valor de um elemento de formulário foi alterado e o foco do usuário saiu do elemento.
