Variáveis Null e Undefined

No JavaScript, as variáveis podem ter dois valores especiais que indicam a ausência de um valor: null e undefined. Embora possam parecer semelhantes, eles representam conceitos ligeiramente diferentes e é crucial entender a distinção para escrever um código robusto.

  • Undefined

    • O valor undefined indica que uma variável foi declarada, mas ainda não foi atribuída a um valor. É o estado inicial padrão de qualquer variável que você cria sem inicializá-la.

  • Null

    • O valor null é um valor de atribuição que significa "nenhum valor" ou "vazio". Ele é um valor intencional que você atribui a uma variável para indicar que ela não contém um objeto ou qualquer outro valor válido.
