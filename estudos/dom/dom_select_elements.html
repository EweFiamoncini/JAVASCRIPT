DOM API - Seleção de elementos

A API do DOM nos dá diferentes métodos no objeto global document para encontrar elementos na sua página HTML. A escolha do método depende de como você quer selecionar o elemento: por ID, por classe, pela tag, ou usando um seletor CSS mais complexo.

  • document.getElementById('id-do-elemento'): Se você quer pegar um elemento único, use o id. É o método mais rápido e direto, porque cada id em uma página deve ser exclusivo.

  • document.getElementsByTagName('sua-tag'): Se você quer encontrar todos os elementos de uma tag específica, como div, p ou a.

  • document.getElementsByClassName('nome-da-classe'): Para selecionar um ou mais elementos que compartilham a mesma classe CSS.

  • document.querySelector('seletor-css'): Retorna o primeiro elemento que corresponder ao seletor.

  • document.querySelectorAll('seletor-css'): Retorna todos os elementos que correspondem ao seletor, em uma lista que pode ser iterada (chamada NodeList).

Qual Usar?

  • Use getElementById() para seleções rápidas e garantidas de um único elemento.
  • Use querySelectorAll() na maioria dos casos, especialmente quando precisar de flexibilidade. Ele é o método mais versátil e moderno.
    • Use querySelector() se você só precisar do primeiro elemento.
