DOM API - Manipulando classes de elementos

Para manipular classes de elementos HTML com JavaScript, a melhor e mais recomendada forma é usar a propriedade classList. Essa abordagem é mais limpa e eficiente do que manipular a string de classes diretamente com className.

A propriedade classList fornece uma série de métodos úteis para adicionar, remover e alternar classes de forma fácil e segura.
