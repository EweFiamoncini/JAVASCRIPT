Estrutura de Repetição - While

As estruturas de repetição, também conhecidas como loops, são ferramentas essenciais em qualquer linguagem de programação, incluindo o JavaScript. Elas permitem que um bloco de código seja executado repetidamente, economizando tempo e esforço, especialmente quando você precisa realizar a mesma tarefa várias vezes.

Em vez de escrever a mesma linha de código 100 vezes, você pode usar um loop para que ele faça isso por você, de forma automática e eficiente.

While

O while loop é usado quando você não sabe de antemão quantas vezes o loop precisa ser executado. Ele continua a rodar enquanto uma condição específica for verdadeira.

while(condicao) {
//codigos
}
