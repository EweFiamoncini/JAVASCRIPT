Switch

O que é comando switch?

Em programação, a instrução switch é usada para executar diferentes blocos de código com base no valor de uma variável ou expressão, oferecendo uma alternativa ao uso prolongado de instruções if-else.

var opcao = 2

switch (opcao) {
 case 1:
  //trecho de código que sera executado
  break

 case 2:
  //techo de codigo que sera executado
  break

 default:
  //techo de codigo que sera executado
  break
}
