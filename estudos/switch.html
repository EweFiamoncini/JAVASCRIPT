Switch

O que é comando switch?

A instrução switch em JavaScript é uma alternativa concisa e eficiente para múltiplas instruções if...else if...else que verificam o mesmo valor.
Ela compara uma expressão com uma série de valores possíveis e executa o bloco de código correspondente ao valor que coincidir.

A sintaxe básica de um bloco switch é a seguinte:

var opcao = 2

switch (opcao) {
 case 1:
  // Bloco de código a ser executado se 'opcao' for igual a '1'
  break

 case 2:
  // Bloco de código a ser executado se 'opcao' for igual a '2'
  break

 default:
  // Bloco de código a ser executado se nenhum dos valores acima coincidir
  break
}
